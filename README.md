# Ёлка — прототип системы

Статическое веб‑приложение с разделением ролей:

- Диспетчер
- Гражданский
- Силовые структуры
- Экстренные службы

## Запуск локально

Просто откройте в браузере файл `index.html`.

## GitHub Pages

1. Создайте репозиторий и загрузите **все файлы из корня** этого проекта:
   - `index.html`, `dispatcher.html`, `citizen.html`, `forces.html`, `rescue.html`
   - `styles.css`, `script.js`, `favicon.svg`
   - папку `img/`

2. В репозитории GitHub:
   - Зайдите в **Settings → Pages**
   - В разделе **Build and deployment** выберите:
     - Source: **Deploy from a branch**
     - Branch: **main** / **root**
   - Сохраните.

3. Через 1–2 минуты сайт будет доступен по адресу:

   ```
   https://<ваш_логин>.github.io/<имя_репозитория>/
   ```

Главная страница — `index.html`.
